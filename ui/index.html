Docksmith

A simplified Docker-like build and runtime system built from scratch in Python, with deterministic layer caching, content-addressed storage, and Linux-rooted isolation.

Build
python -m docksmith.cli build -t myapp:latest sample-app
Run
python -m docksmith.cli run myapp:latest
Base Import
python scripts/import_base_image.py --name python-base --tag 3.11 --tar /path/to/rootfs.tar
